# Compound Interest Equations

$A = P \left(1 + \frac{r}{n}\right)^{nt}$

where:\
$A$ = amount   \
$P$ = principal \
$r$ = rate (in decimal, e.g. 8\% is 0.08) \
$n$ = number of times compounded per year \
$t$ = number of years

If it is compounded annually, then $n=1$. This simplifies the equation to:

$\boxed{A = P \left(1 + r\right)^t}$

The equation can be rearranged given the known and unknown quantities.

**Amount**, given the principal and rate.

$A = P \left(1 + r\right)^t$

**Principal**, given the amount and rate.

$P = \frac{A}{\left(1 + r\right)^t}$

**Rate**, given the amount and principal.

$r = \left(\frac{A}{P}\right)^\frac{1}{t}-1$
